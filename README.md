#Med-Annotate: Oral Health Management System
Overview

Med-Annotate is a web-based Oral Health Management System designed to assist in the detection, annotation, and management of oral health conditions. The system integrates modern web technologies with intelligent processing to help users and healthcare professionals analyze oral images efficiently.

---

## Features

*  Upload and analyze oral health images
*  AI-based detection and annotation
*  Structured data handling and visualization
*  Secure backend with API integration
*  User-friendly web interface

---

##  Tech Stack

### Frontend

* React.js
* HTML, CSS, JavaScript

### Backend

* Node.js
* Express.js

### Other Tools

* Puppeteer (for automation/image processing)
* REST APIs

## ⚙️ Installation & Setup

### 1️ Clone the repository

```bash
git clone https://github.com/sairevanth286/Med-Annotate-Oral-Health-Management-System.git
cd Med-Annotate-Oral-Health-Management-System
```

---

### 2️ Install dependencies

#### Backend

```bash
cd backend
npm install
```

#### Frontend

```bash
cd frontend
npm install
```

---

### 3️⃣ Run the application

#### Start backend

```bash
cd backend
npm start
```

#### Start frontend

```bash
cd frontend
npm start
```

---
